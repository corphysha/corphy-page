## Why Astro?

I chose **Astro** for this blog because:

1. **Static by default** — zero JavaScript shipped to the client unless needed
2. **Content Collections** — type-safe Markdown/MDX with Zod validation
3. **View Transitions** — buttery-smooth page navigation
4. **Markdown-native** — write posts in MDX with embedded components

The entire site is built with:

```ts
// astro.config.mjs
import { defineConfig } from "astro/config";
import sitemap from "@astrojs/sitemap";
import mdx from "@astrojs/mdx";

export default defineConfig({
  site: "https://corphysha.github.io",
  base: "/corphy-page",
  integrations: [sitemap(), mdx()],
});
```
